# intellij shortcuts

Find in all files
```aidl
cmd + shift + f
```
Find in the current file

```aidl
cmd + f
```

Recent files
```aidl
cmd + e
```

Search by file name

```aidl
cmd + shift + o
```

Search in classes

```aidl
cmd + o
```

Search everywhere (Joker)
```aidl
shift 2 times
```
Move back 
```aidl
cmd + { 
```
Move forward
```aidl
cmd + }
```
Go to method definition
```aidl
cmd + b or cmd + click

```
